# Medieval Europe Historical Atlas — Step 1

This is the first project scaffold for the interactive medieval Europe atlas.

## What works

- Year slider from 500 to 1500 CE
- Embedded offline Europe basemap
- Simplified benchmark historical snapshots
- Religious-control colors
- Pope panel changes by year
- Major historical events near the selected year
- Important people involved in those events
- Clickable regions
- Zoom and pan

## How to open

Best:
1. Upload this folder to Netlify Drop, GitHub Pages, or any static web host.
2. Open the generated URL in Safari/Chrome.

On Mac/Windows:
- Open `index.html` directly in Chrome, Edge, Firefox, or Safari.

On iPad:
- Local HTML files often block JavaScript depending on the app.
- Uploading to Netlify/GitHub Pages is recommended.
